# Application Mobile - SIGEANSUD

Application Android (Java) pour les soignants du parc.

## Fonctionnalites

- Authentification du soignant
- Consultation des animaux des especes affectees
- Saisie des heures et quantites des repas donnes
- Stockage local SQLite pour le soignant

## Prerequis

- Android Studio
- SDK Android minimum API 26
