# Back-Office - SIGEANSUD

Application web Symfony 7 pour les gestionnaires du parc.

## Fonctionnalites

- Gestion des soignants et de leurs especes
- Gestion des menus par espece
- Consultation des repas donnes

## Installation

```bash
composer install
cp .env .env.local  # configurer DATABASE_URL
symfony server:start
```
